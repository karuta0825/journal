org-bufferで有効なプロパティを取得する  

```elisp
(org-buffer-property-keys)
```

pointにあるエントリーのプロパティを取得する  

```elisp
(org-entry-properties)
```

バッファ内の全エントリーからある関数を実行する  

```elisp
(org-map-entries (lambda() (princ (org-entry-properties))))
```

```elisp
(org-export-get-environment)
```
